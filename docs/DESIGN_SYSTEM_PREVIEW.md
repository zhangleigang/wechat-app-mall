## 🎯 使用建议

### 1. 颜色使用
- 主题色：重要操作、强调内容
- 成功色：成功提示、完成状态
- 警告色：警告提示、注意事项
- 危险色：错误提示、删除操作

### 2. 间距使用
- xs/sm：标签、图标间距
- md/lg：卡片内边距、元素间距
- xl/2xl：区块间距、页面边距

### 3. 字体使用
- xs/sm：辅助文字、标签
- md/lg：正文、按钮
- xl/2xl：标题、重要信息

### 4. 阴影使用
- xs/sm：轻微层次
- md/lg：卡片、弹窗
- xl/2xl：强调、悬浮
